# Update Ligand and Receptor Database
To enhance the ligand-receptor database, we compiled interaction pairs from:
- [`NeuronChat`](https://github.com/Wei-BioMath/NeuronChat),
- [`ICELLNETdb`](https://github.com/soumelis-lab/ICELLNET),
- [`iTALK`](https://github.com/Coolgenome/iTALK),
- [`SingleCellSignalR`](https://github.com/SCA-IRCM/SingleCellSignalR),
- [`CellTalkDB`](https://github.com/ZJUFanLab/CellTalkDB),
- [`Celltalker`](https://github.com/CilloLaboratory/celltalker),
- [`NATMI`](https://github.com/forrest-lab/NATMI)
